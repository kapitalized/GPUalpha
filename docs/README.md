# Documentation

This folder contains project documentation organized by category.

## Structure

- **Root** - Active reference documentation (keep in project root)
- **archive/** - Completed/resolved documentation (for historical reference)

## Active Documentation (Root)

These files are kept in the project root for easy access:

- `README.md` - Main project documentation
- `ENV_TEMPLATE.md` - Environment variables template
- `VERCEL_ENV_SETUP.md` - Vercel deployment setup guide
- `SECURITY_ENV_GUIDE.md` - Security best practices
- `CRON_SETUP.md` - Cron job configuration
- `GPU_DATA_SOURCES.md` - GPU data source information
- `TIMESCALE_GUIDE.md` - TimescaleDB setup guide
- `TIMESERIES_SETUP.md` - Time-series data setup
- `REPOSITIONING_STRATEGY.md` - Product positioning strategy
